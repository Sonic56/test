# DSi Sound — Android App

A faithful recreation of the Nintendo DSi Sound app for Android, with all original assets.

---

## ✅ How to get the APK (no Android Studio needed)

### Step 1 — Create a free GitHub account
Go to **https://github.com** and sign up (it's free).

### Step 2 — Create a new repository
1. Click the **+** button (top right) → **New repository**
2. Name it `DSiSound`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Enable Actions permissions
1. In your new repo, click **Settings** tab
2. Click **Actions** → **General** (left sidebar)
3. Under "Workflow permissions" select **Read and write permissions**
4. Click **Save**

### Step 4 — Upload the project files
1. On your repo page, click **uploading an existing file**
2. Drag and drop **all the files and folders** from inside the DSiSound ZIP
3. Scroll down, click **Commit changes**

### Step 5 — Watch it build
1. Click the **Actions** tab in your repo
2. You'll see **"Build DSi Sound APK"** running
3. Wait ~5 minutes for it to finish (green checkmark)

### Step 6 — Download your APK
- Click **Releases** on your repo homepage → download `app-debug.apk`
- OR: Click the finished Actions run → scroll to **Artifacts** → download

### Step 7 — Install on your phone
1. Transfer the APK to your Android phone
2. Settings → Security → Install unknown apps → Allow
3. Tap the APK to install

## Features
- Original DSi Sound parakeet icon
- 11 animated visualizer backgrounds
- Dancing chick sprites
- Pitch & Speed sliders
- Full playback controls
- Browses all music on your device
